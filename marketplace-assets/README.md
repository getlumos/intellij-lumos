# JetBrains Marketplace Assets

This directory contains all assets needed for the JetBrains Marketplace listing.

## Files

### Plugin Icon
- **plugin-icon.png** (512x512) - Main plugin icon for marketplace listing

### Screenshots
1. **screenshot-1-syntax.png** (352 KB) - Syntax highlighting showcase
2. **screenshot-2-completion.png** (31 KB) - Auto-completion feature
3. **screenshot-3-hover.png** (53 KB) - Hover documentation tooltip

## Usage

When uploading to JetBrains Marketplace (https://plugins.jetbrains.com):

1. **Plugin Icon**: Upload in "Edit Plugin" → "Icon" section
2. **Screenshots**: Upload in "Edit Plugin" → "Screenshots" section in this order

## Requirements

- **Icon**: 512x512 PNG
- **Screenshots**: 1280x800 or 1920x1080 PNG (ours are optimized for marketplace)

All files are ready for upload!
